# CO2 — IT-OT Access Workflow Automation

This repository contains the BPMN workflow representing secure IT-OT access control.

## Files
- `it-ot-access.bpmn` — Executable BPMN diagram

## Architecture Context
Derived from CO1 OT–DMZ–IT architecture.
Implements governance and secure access control matching ISA-62443 workflow concepts.

## Execution Pipeline
- Designed in Camunda Web Modeler (browser)
- Deployed in Camunda SaaS workflow engine
- Verified in Camunda Operate dashboard

## Summary
This project demonstrates how an OT security architecture can map to an automated BPMN workflow for controlled IT-OT access approvals.
